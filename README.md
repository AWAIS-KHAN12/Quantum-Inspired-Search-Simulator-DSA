# Quantum-Inspired Search Simulator

A powerful and interactive Data Structures & Algorithms project built in C++ that demonstrates the comparison between classical search algorithms and quantum-inspired search techniques. This application simulates the core principles of Grover's Algorithm, showcasing how quantum computing concepts can inspire faster search solutions.

---

## 🚀 Features

- Linear Search implementation
- Binary Search implementation
- Quantum-Inspired Search simulation
- Grover's Algorithm concepts
- Max Heap Priority Queue
- Interactive GUI
- Robust input validation
- Performance comparison between algorithms
- Probability amplification visualization

---

## 🧠 Algorithms Implemented

### Linear Search
- Time Complexity: **O(n)**
- Sequentially checks each element.

### Binary Search
- Time Complexity: **O(log n)**
- Requires sorted input.

### Quantum-Inspired Search
- Time Complexity: **O(√n)**
- Simulates Grover's Search Algorithm using:
  - Amplitude Initialization
  - Oracle Phase Inversion
  - Diffusion Operator
  - Probability Measurement
